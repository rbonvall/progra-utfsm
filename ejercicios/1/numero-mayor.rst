Número mayor
-------------

Realizar un programa que permita
determinar el número mayor perteneciente
a un conjunto de *n* números,
donde tanto el valor de *n* como
el de los números deben ser ingresados
por el usuario.

* Comportamiento

::
   
   Ingrese n: 4
   Ingrese número: 23
   Ingrese número: -34
   Ingrese número: 0
   Ingrese número: 1
   El mayor es 23

::

   Ingrese n: 5
   Ingrese número: -34
   Ingrese número: -12
   Ingrese número: -1
   Ingrese número: -100
   Ingrese número: 0
   El mayor es 0
